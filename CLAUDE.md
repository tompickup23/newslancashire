# News Lancashire — Claude Code Context

## Overview
Local news for Lancashire. Community journalism and local reporting.

**Stack**: HTML | CSS | JavaScript
**Hosting**: GitHub Pages
**Branch**: main

## Key Patterns
- Static HTML site with news articles
- Mobile-first responsive design
- SEO-optimised article pages
- Social sharing metadata (Open Graph, Twitter cards)

## Rules
- Accurate, factual reporting
- British English throughout
- Semantic HTML (`<article>`, `<header>`, `<time>`)
- All images need alt text and captions
- Mobile-first responsive design
- Fast page loads — optimise images (WebP, lazy-load)
- Each article needs proper `<title>` and meta description
