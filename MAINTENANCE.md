# News Lancashire - Quick Start & Maintenance Guide

## 🚀 Quick Commands (After SSH to Thurinus)

```bash
# Easy menu-driven updates
~/newslancashire/scripts/easy-update.sh

# Or run individual commands:

# 1. Add new original article
hugo new content original/my-article.md --editor nano

# 2. Build and deploy site
cd ~/newslancashire/site && hugo --minify && sudo chown -R www-data:www-data public

# 3. Run crawler manually
python3 ~/newslancashire/scripts/crawler.py

# 4. Export articles for AI summary
~/newslancashire/scripts/generate-summaries.sh
```

## ✏️ Editing Content

### Adding Original Articles (Recommended)
```bash
# Via easy menu
~/newslancashire/scripts/easy-update.sh
# Then choose option 1

# Or manually
nano ~/newslancashire/site/content/original/your-article.md
```

**Front matter format:**
```yaml
---
title: "Your Article Title"
date: 2026-02-05T10:00:00Z
draft: false
source: "Tom Pickup"
---

Write your article in **Markdown** here.
```

## 🔄 Regular Maintenance

### Daily (Optional)
- Check if crawler ran: `tail ~/newslancashire/logs/crawler.log`
- Review new articles for AI summarization

### Weekly
- Write 1-2 original articles (builds credibility)
- Review and edit AI summaries
- Check site analytics

### Monthly
- Update RSS sources if needed: `nano ~/newslancashire/scripts/crawler.py`

## 🆘 Troubleshooting

### Site not updating after changes?
```bash
# Rebuild manually
cd ~/newslancashire/site
hugo --minify
sudo chown -R www-data:www-data public
sudo systemctl reload nginx
```

### Crawler not working?
```bash
# Check logs
tail ~/newslancashire/logs/crawler.log

# Run manually to see errors
python3 ~/newslancashire/scripts/crawler.py
```

### Permission denied errors?
```bash
sudo chown -R ubuntu:ubuntu ~/newslancashire
sudo chown -R www-data:www-data ~/newslancashire/site/public
```

### Site not loading?
```bash
# Check nginx status
sudo systemctl status nginx

# Check if site built
ls -la ~/newslancashire/site/public/
```
